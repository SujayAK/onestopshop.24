🎯 5-STEP SETUP:

STEP 1: Create Supabase Table
──────────────────────────────
1. Go to Supabase Dashboard
2. Click SQL Editor
3. Run this:

CREATE TABLE stock_clearance_products (
  id BIGINT PRIMARY KEY GENERATED ALWAYS AS IDENTITY,
  sr_no INTEGER,
  name TEXT NOT NULL,
  price NUMERIC(10, 2) NOT NULL,
  category TEXT CHECK (category IN ('Tops', 'Bottom', 'Dresses')),
  size TEXT,
  colors TEXT,
  stock INTEGER DEFAULT 0,
  description TEXT,
  image_url TEXT NOT NULL,
  active BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_category ON stock_clearance_products(category);
CREATE INDEX idx_active ON stock_clearance_products(active);

💡 PRO TIPS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Image Optimization
   • Use TinyPNG.com to compress before upload
   • Target: 100-300 KB per image
   • Resize to 800x800 pixels
   • Use 75% JPEG quality

2. Avoid Supabase Storage Charges
   • ✓ Upload images to ImgBB (free)
   • ✗ Don't store images in Supabase
   • ✓ Link CSV to ImgBB URLs
   • Keep Supabase for database only

3. Testing with Sample Data
   • Start with 5-10 products
   • Test filters work
   • Then bulk import all

4. Regular Updates
   • CSV changes: Easy to update and re-import
   • Stock levels: Update stock column anytime
   • Active/Inactive: Toggle with true/false
   • Price changes: Adjust price column

❓ QUICK TROUBLESHOOTING:

Q: Products not showing?
A: 1. Check table is created
   2. Verify CSV imported correctly
   3. Check active: true in CSV
   4. Hard refresh browser (Ctrl+F5)

Q: Images not loading?
A: 1. Test image URL directly in browser
   2. Verify ImgBB URL is complete
   3. Check CDN is accessible
   4. Look at Network tab in DevTools

Q: Slider not working?
A: 1. Clear browser cache
   2. Hard refresh (Ctrl+F5)
   3. Check CSS file loaded
   4. Look at console for errors

Q: Filters not applying?
A: 1. Verify Supabase is configured
   2. Check internet connection
   3. Open DevTools console for errors
   4. Try resetting filters
